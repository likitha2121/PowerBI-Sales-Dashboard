# Power BI Sales Dashboard

## Project Overview

This project presents an interactive **Sales Performance Dashboard** built using Power BI.
The dashboard helps in analyzing sales data, identifying trends, and making data-driven decisions.



## Objectives

* Analyze overall sales performance
* Track monthly sales trends
* Identify top-performing products and regions
* Compare targets vs actual sales
* Provide interactive filtering for better insights



## Dataset

* File: `powerbi_dataset_10000x21.csv`
* Records: ~10,000 rows
* Columns: 21 features including:

  * Order ID
  * Order Date
  * Customer Name
  * Region
  * Product Category
  * Sales Amount
  * Profit
  * Quantity

## Tools & Technologies Used

* Power BI (Data Visualization)
* Power Query (Data Cleaning & Transformation)
* DAX (Calculated Measures)
* Git & GitHub (Version Control)


##  Dashboard Features

### 1. KPI Cards

* Total Sales
* Total Profit
* Total Orders

### 2. Sales Trend Analysis

* Line chart showing monthly sales trend

### 3. Category-wise Sales

* Bar chart comparing product categories

### 4. Region-wise Performance

* Map / Bar chart for regional sales distribution

### 5. Target vs Actual

* Gauge chart showing performance against target

### 6. Sales Funnel

* Funnel chart showing sales stages

### 7. Scatter Plot

* Profit vs Sales analysis

### 8. Slicers (Filters)

* Region
* Category
* Date
